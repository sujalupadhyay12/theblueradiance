# Images Directory

This directory contains all image assets for The Blue Radiance website.

## Files

- **favicon.ico** - Website favicon (16x16px or 32x32px)
- **og-image.jpg** - Open Graph image for social sharing (1200x630px recommended)

## Guidelines

### Favicon (favicon.ico)
- Format: ICO or PNG
- Size: 16x16, 32x32, or 64x64 pixels
- Design: Company logo or anchor symbol
- Color: Deep navy blue (#001F3F)

### OG Image (og-image.jpg)
- Format: JPG/PNG
- Size: 1200x630 pixels minimum
- Aspect Ratio: 1.91:1
- Shows: Company branding with services
- Used for: Social media sharing

## Creating Favicon

1. Design a small icon (256x256px)
2. Convert to ICO format using online tools
3. Place in root directory
4. Reference in HTML: `<link rel="icon" href="favicon.ico">`

## Image Optimization Tips

- Compress JPG/PNG before upload
- Use online tools like TinyPNG or ImageOptim
- Recommended sizes: 100-200KB max
- PNG for transparency, JPG for photos
